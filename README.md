## 搭子匹配系统（Buddy Match)

**此仓库为搭子匹配系统后端源码**

主旨为帮助大家找到各种搭子的移动端 APP 风格网站，实现了用户管理、按标签搜索用户、

推荐相似用户、组队等功能

## 技术选型

- SpringBoot框架
- MySQL数据库
- MyBatis-Plus、MyBatis X 动生成代码
- Redis缓存、Redis分布式登录
- Redisson分布式锁
- Spring Scheduler定时任务
- Apifox 接口管理平台
- 相似度匹配算法
